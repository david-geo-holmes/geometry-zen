Geometry Zen

Educational Computational Physics, Modeling and Geometry


Geometry Zen is a free, online, open-source, and collaborative tool for Educational Computational Physics, Modeling and Geometry combining programming, 3D visualization, Physical Units, and Geometric Algebra - the Unified Mathematical Language for Physics and Engineering in the 21st Century.

Browse Examples

Philosophy

The purpose of this software is to help students achieve mastery of a wide variety of topics in physics, mathematics and engineering (STEM) through the authentic construction of executable (software) models.

Design Goals

  • Pedagogy: Facilitate learning and mastery in a non-threatening way by allowing students to experiment and experience.
  • Ease of Setup: Minimize the need for software engineering skills.
  • Ease of Use: Minimize the need to refer to external documentation. Automatically detect programming errors.
  • Performance: Be able to execute simulations of appropriate complexity within several seconds.
  • Mathematical Notation: Support usual arithmetic operators as well as those used in Geometric Algebra and with appropriate precedence.
  • Open Source: Make all the source code available to the developer community and engage users in the product roadmap.
  • Data Ownership: Ensure that artifacts created by users are owned by the users and available independent of the application.

Features

  • Languages: Choose from TypeScript, JavaScript, CoffeeScript or Python.
  • Smart Editing: Editor performs syntax checking as you type and provides context-sensitive documentation.
  • Operator Overloading: Allows you to write code using natural math operators, even for non-primitive objects such as complex numbers and vectors, as well as your own classes.
  • Operator Precedence: Allows you to minimize parenthesis clutter by having a well-defined order of operators.
  • Visualization Libraries: Popular libraries such as three.js (3D geometry), JXG (2D geometry), d3 (data visualization), easel (2D rendering) are automatically available and feature integrated context-sensitive documentation and type checking.
  • Computation Libraries: Libraries for Vector and Geometric Algebra, Complex Numbers and Quaternions. Support for SI units of measure and dimensions.
  • GitHub Integration: All user programs are stored in the personal (GitHub) account of the user. You will therefore need to have a (free) GitHub account to store your work.

Editor Note - Roadmap

Thank You for considering GeometryZen to make your courses and assignments more engaging and effective!

I strongly encourage you to explore TypeScript (JavaScript with optional typing) as the programming language when looking for a friendly editing experience with type-checking, context-sensitive help and code-completion.

The immediate roadmap consists of providing more curated examples, better documentation support, and exploiting GitHub Gists as the basis for assignments.

I hope you will become part of the GeometryZen community!
